M4Loc integration tools installation 
====================================

PREREQUISITES
=============
Unix-based OS
Perl v 5.10.0 or higher
CPAN libraries:
    IPC::Run
    IPC::Open2
    XML::LibXML::Reader
    HTML::Entities
Okapi M22 or higher: http://okapi.opentag.com/
    Okapi executable directory needs to be added to PATH environment variable
Moses v1.0 or higher: http://www.statmt.org/moses
    Moses /bin directory and all directories under /scripts need to be added 
    to PATH environment variable

INSTALLATION
============
1. Go to http://code.google.com/p/m4loc/
2. Click on Source tab
3. Click on Browse
4. Download ZIP file using link "Download *zip*"
5. Extract ZIP file
6. Add m4loc/xliff directory to PATH environment variable
